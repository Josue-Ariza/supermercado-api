# Supermercado API - Práctica de Spring Boot 🛒

Este proyecto es una API REST desarrollada como parte de mi formación técnica en **Spring Boot**. El objetivo principal fue poner en práctica el manejo de persistencia de datos y la creación de servicios web con Java.

##  Objetivos de la Práctica
* Implementar el framework **Spring Boot**.
* Gestión de base de datos relacional con **Spring Data JPA**.
* Creación de Endpoints para operaciones CRUD.
* Aplicación de la arquitectura por capas (Controller, Service, Repository).

##  Tecnologías y Herramientas
* **Java**
* **Spring Boot**
* **Maven** (Gestor de dependencias)
* **Postgresql** (Persistencia de datos)

##  Estructura del Proyecto
El código se organiza siguiendo el estándar de la industria:
* **Persistence:** Mapeo de tablas y repositorios.
* **Domain:** Lógica de negocio y modelos.
* **Web:** Controladores para exponer la API.
